# 2026 高考志愿预选项目 Agent

更新时间：2026-06-10  
适用对象：陕西/西安考生，2026 年高考，物化生，估分 640-660，向下预留至 630。

## 1. 项目目标

在 2026 年正式成绩、一分一段表和招生计划公布前，先用 2025 年陕西物理类录取数据建立一套可持续迭代的学校和专业备选池。当前不是定稿，而是先做“宽表 + 初筛 + 讨论框架”，后续按真实位次快速收敛到冲、稳、保志愿方案。

核心原则：

1. 以陕西物理类位次为主，不直接迷信裸分。
2. 先保可接受专业底线，再看学校牌子；但承认 985/211/双一流在企业筛简历、保研、央国企和大厂初筛中有现实价值。
3. 优先西安高校；其次一线/强产业城市；位置不是硬约束。
4. 优先 985，其次头部 211；中外合作/高收费项目可作为高性价比备选。
5. 不倾向医学、师范、农林、法学、财经；文科类可作为研究生/辅修/职业转向，不作为本科主线。

主线是西交大、西工大、电子科大、北邮强专业。西安优先时，西交大软件、医学电子信息、智能感知；西工大电子、计算机、航空航天都可以进入冲刺池。需要注意，西交大很多顶尖电类/计算机类可能高于 660，不能只看学校，要看能进哪个专业类。

### 如果最终接近 650

最有讨论价值的是西工大计算机、电子信息、自动化、航空航天；西交大智能制造、航天航空、智慧能源；电子科大计算机/软件/电子；北邮普通计算机、通信、电子、AI、网安。这个分段要重点解决“西安 985 vs 外地强专业 211/985”的取舍。

### 如果最终接近 640

要把“专业适配”权重提高。西电的电子信息/计算机实验班，北邮普通计算机/通信/AI/网安，西工大部分软件/智能制造/水下智能，外地 985 的电子、计算机、电气、自动化都可作为稳妥池。这个分数不宜为了学校牌子进入明显不喜欢的医学、基础理科、材料化工或财经方向。

### 如果最终接近 630

核心目标是保住 985/头部 211 中仍可接受的工科专业。重庆大学、湖南大学、中南大学、四川大学、山东大学、北京交通大学、南京航空航天等要继续补全。西电电子工程/集成电路/通信也有很强安全价值。若正式位次低于 630 档，再启用长安大学、西北大学等本地 211 安全池。

## 8. 专业选择建议

不要因为这次数学、物理、化学发挥不理想，就简单否定工科。工科和纯理科不同，很多方向更强调工程实现、项目能力、工具链、系统理解和持续学习。但电子、通信、自动化、电气、航空航天、AI、计算机都离不开高数、线代、概率、大学物理、电路/信号/控制/算法，不能把它们理解成“轻松的应用专业”。

当前更建议采用“应用型工科 + 数字能力 + 行业场景”的组合。

| 优先级 | 方向 | 代表专业 | 为什么适合 | 主要风险 |
|---|---|---|---|---|
| 高 | 计算机/软件/网安 | 计算机、软件工程、网络空间安全、数据科学 | 就业面宽，可迁移能力强；画画兴趣可转向交互、前端、可视化、AIGC 工具 | 竞争强，需要持续编程和项目 |
| 高 | 电子信息/通信/集成电路 | 电子信息、通信、微电子、集成电路 | 国家战略和产业需求明确，西安/成都/北京都有强校 | 数学、电路、信号课程硬 |
| 高 | 自动化/机器人/智能制造 | 自动化、机器人工程、智能制造、测控 | 制造业升级、机器人、工业软件长期需要 | 控制、电路、机械基础都有压力 |
| 中高 | 医工交叉 | 医学电子信息、生物医学工程、智能医学工程 | 兼顾物化生背景和生物兴趣，不等同于临床医学 | 本科出口差异大，常需要读研 |
| 中高 | 低空/航空航天智能化 | 低空技术、航空航天、无人系统、飞控 | 政策和产业热度高，西工大/北邮相关专业出现 | 力学、控制、飞设硬度高，行业有周期 |
| 中 | 数字媒体/交互设计技术 | 智能交互设计、数字媒体技术、工业设计+计算机 | 契合画画兴趣，可形成审美+技术差异化 | 学校专业内涵差异大，不能只看名字 |
| 谨慎 | 材料/化工/纯生物工程 | 材料、化工、生物工程 | 强校读研可行 | 本科就业分化大，不作为当前主线 |
| 谨慎 | 纯数学/物理/化学 | 基础理科、拔尖班 | 学术路线可行 | 与当前发挥和职业目标不完全匹配 |

## 9. 结合孩子特点的专业偏好排序

首轮建议按下面顺序建池：

1. 计算机/软件/网安/数据：就业面和可迁移性最强，适合未来 AI 工具时代。
2. 电子信息/通信/集成电路：适合西电、电子科大、北邮、西工大等强校，但课程硬。
3. 自动化/机器人/智能制造/测控：比纯理论更工程化，适合制造业升级。
4. 医工交叉/医学电子信息：利用生物兴趣，但避开临床医学长周期。
5. 智能交互/数字媒体技术：把画画兴趣转为技术产品能力，重点看是否在强计算机学院内。
6. 低空/航空航天/无人系统：西工大优势明显，但要确认孩子是否接受力学和工程强度。

## 10. 未来 5-10 年判断

更可靠的选择不是追单个热门词，而是让本科专业提供一个能跨行业迁移的技术底座。未来 5-10 年值得重点关注的能力组合：

| 能力/赛道 | 对应本科方向 | 判断 |
|---|---|---|
| AI 工具化和软件化 | 计算机、软件、数据、网安 | 几乎所有行业都会被 AI 改造，但真正有价值的是能落地系统的人 |
| 端侧智能和物联网 | 电子信息、通信、嵌入式、自动化 | 6G、车联网、工业互联网、机器人都需要软硬结合 |
| 智能制造和机器人 | 自动化、机器人、智能制造、测控 | 中国制造业升级确定性高，应用场景多 |
| 低空经济和无人系统 | 航空航天、飞控、通信、低空技术 | 政策热度高，但本科要选有真实学科和实验平台的学校 |
| 医疗健康数字化 | 医工交叉、医学电子、AI 医疗影像 | 人口老龄化和医疗效率提升是长期需求，常需读研 |
| 能源电力和储能 | 电气、能源动力、储能、智能电网 | 就业相对稳，央国企路径清晰，但课程不轻 |
| 人机交互和数字内容 | 交互设计、数字媒体技术、AIGC 工具链 | 喜欢画画的优势可转为产品、体验、可视化、内容生成方向 |

专业避坑原则：

1. 只看专业名字里的“人工智能”“智能”“低空”不够，要看所在学院、课程、保研、就业和实验平台。
2. 大类招生要查清分流规则，避免入校后被分到不接受的方向。
3. 软件工程、工业软件、部分试验班要查学费和校区。
4. 强校冷门专业如果孩子不接受，不应只为了 985 标签硬报。
5. 真正的就业竞争力来自“学校层次 + 专业底座 + 项目/实习 + 英文/表达 + AI 工具能力”。

## 13. 已确认偏好与仍需确认问题

已确认：

1. 编程可以作为大学四年的核心能力训练。家庭判断是，AI 时代并不会降低编程基本功的重要性，反而会把人分成底层算法、系统架构/方案能力、普通工具使用等层次；计算机/软件/AI 工程方向维持高权重。
2. 可以接受电子、电路、信号、控制等工科课程；电子信息、通信、自动化、测控、集成电路、机器人、智能制造均可进入主池。
3. 接受读研。孩子喜欢生物，当前更倾向“本科工科底座 + 研究生生物医药/医工交叉/AI 医疗/计算生物”的长期路线。
4. 非主校区、较高学费可考虑，但前提是学校和专业优势明确。例如强校威海、沙河等不直接排除，但要标注毕业证、校区资源、学费和就业差异。
5. 外地更强专业/城市略占优，但西安本地资源同样重要，两者权重接近。因此志愿池采用“双核心”：西安 985/211 资源线 + 外地强专业/强城市线。
6. 画画主要是个人爱好，不走艺术类主线。可作为科技艺术、交互设计、产品审美、可视化表达能力的辅助项。

仍需确认：

1. 孩子是否真正愿意持续做编程项目，包括数据结构、操作系统、网络、工程实践，而不只是接受“AI”概念。
2. 对电子信息方向，要进一步确认是否接受电路、模电、信号与系统、电磁场等硬课强度。
3. 对低空/航空航天方向，要确认是否接受力学、飞控、工程物理和行业周期。
4. 对医工交叉/生物医学工程，要确认是否能接受读研周期和本科阶段偏工程而非纯生物兴趣。
5. 对非主校区/高学费专业，要等具体志愿表中逐项核对校区、学费、转专业、毕业证和就业数据。

## 14. 当前结论摘要

现阶段最值得优先讨论的不是“理科没考好所以不能工科”，而是把工科拆成不同强度和不同出口：计算机/软件偏项目和自学，电子通信偏数学电路信号，自动化偏控制和工程系统，航空航天偏力学和工程物理，医工交叉偏读研和应用场景。

如果正式成绩在 650 以上，西交大、西工大、电子科大、北邮都应进入核心讨论；如果在 640 左右，西工大部分专业、西电强专业、北邮普通强专业和若干 985 工科是主池；如果在 630 左右，重庆大学、湖南大学、中南大学、四川大学、山东大学、北京交通大学、西电等要承担稳保功能。西北大学、长安大学暂作为“估分明显偏高时的本地 211 安全池”，不是当前主推方向。

## 16. 基于 2026 发布状态的阶段性建议

截至 2026-06-10，本项目不能把任何目标院校的“2026 陕西物理普通类分专业计划数”作为已经确定的数据来使用。当前可执行策略是：

1. 继续用 2025 专业录取位次建立初筛池，不用 2026 章程替代计划数。
2. 对已经发布 2026 专业目录的学校，先更新“可选专业池”和“排除专业池”，但不调整冲稳保梯度。
3. 等 2026 陕西招生计划目录发布后，对每个候选专业做计划增减判断：计划缩减 20% 以上提高风险等级；计划扩招 20% 以上可略微降低风险，但仍以位次为核心。
4. 重点盯本地三校：西交大、西工大、西电。若 650+，先看西交/西工大能否拿到可接受工科；若 640 左右，西电强专业和西工大部分方向仍是核心；若 630 左右，要把外地 985 中段专业和本地 211 安全池做厚。
5. 电子科技大学 2026 目录中“科技与艺术联合学位”“低空技术与工程”“医工交叉”等方向与孩子兴趣有交集，后续只要陕西有计划，应进入讨论表，但暂不高于计算机、电子信息、集成电路、自动化这些主线方向。

下一轮应优先做两件事：

1. 每天复查目标院校查询系统是否新增 2026 年份，尤其是西交大、西工大、西电、电子科大、北邮。
2. 一旦陕西省招生计划汇编发布，把每个候选专业的 2026 计划数、2025 计划数、2025 最低分/位次、专业备注、校区、学费放到同一张最终志愿测算表里。

## 30. 新增硬约束：视力与专业清/二次分流（2026-06-11）

### 视力条件

孩子近视约 475 度和 500 度，矫正视力正常，可接近 1.5。最终判断不能只看度数，要以高考体检表中的裸眼视力、矫正视力、镜片度数、色觉检查和学校招生章程为准。

处理口径：

1. 普通计算机、软件、电子信息、通信、自动化、集成电路等方向不因该近视条件直接排除。
2. 测控技术与仪器、智能仪器、智能感知、生物医学工程、医学电子、飞行器制造等方向要标为“视力/不宜就读待核”，但因矫正视力好，不做简单排除。
3. 飞行技术、航海技术、轮机工程、消防工程、公安类等有裸眼视力、矫正视力或镜片度数硬性要求的方向原则上不作为主推，除非学校章程明确允许。
4. 最终志愿表中每个视力敏感专业必须问招生办：是否限制裸眼视力、矫正视力、镜片度数、色觉，是否影响录取、分流或转专业。

### 中外合作/高收费项目新口径

家庭接受为了更强学校、更好专业和更低录取位次支付较高费用。几十万成本如果能显著改善孩子长期平台和专业出口，可以认为值得。

处理口径：

1. 中外合作/高收费项目不再默认排除，也不在 dashboard 的“隐藏明显不符合偏好”中自动隐藏。
2. 若分数优惠明显，且能进入更强学校、更强专业，可以进入“高性价比备选池”。
3. 必须核清：毕业证、学位证、培养地点、国内外学习年限、总费用、外方课程比例、保研资格、就业质量、转专业限制、是否可调剂。
4. 中外合作不能和普通专业简单混排，要单独标注费用和证书风险；如果只是弱专业换学校名头，仍不提高优先级。

### 专业清和入校后二次分流

这里分两类风险，不能混淆：

1. 专业清/分数清/专业级差：这是投档到学校后，学校安排具体专业的录取规则。专业清更重视专业志愿顺序，专业级差会让后续专业志愿扣分或降低优先级。
2. 入校后二次分流：大类、试验班、卓越班、拔尖班、未来技术学院等入校后，学校再按成绩、志愿、面试、名额等规则分到具体专业。

处理口径：

1. dashboard 的待核事项已统一加入“核招生章程：专业清/分数清/专业级差/调剂规则”。
2. 大类、试验班、卓越、拔尖、未来技术、长空创新班等会额外提示“确认入校后二次分流规则、目标专业名额和不可接受方向”。
3. 自动预填志愿和导出 CSV 会带出这些待核事项。
4. 最终排序时，如果不能确认目标专业名额和分流规则，该项不能作为稳项，只能放在冲刺或待核备选。

1. `plan` 或 `plan2025`：2025 分省分专业计划招生人数。优先来自学校招生查询系统、陕西招生计划汇编、学校官网分专业计划。
2. `admitted2025` / `actualAdmitted2025`：2025 最后实际录取人数。只有学校或省级官方明确公开“录取人数/实际录取人数”时才录入。
3. 不允许默认 `实际录取人数 = 计划人数`。追加计划、同分录取、退档、调剂、预留计划、大类合并拆分都可能导致两者不完全一致。
4. 当前项目已有一部分 A 源计划数，主要是西交大、西电；多数 B 源候选项计划数和实录人数仍待回学校官网或陕西计划汇编复核。
5. 新增 `scripts/generate-plan-admit-audit.mjs`，生成 `data/generated/plan-admit-reconciliation-2025.json` 和 `2025计划与实际录取人数审计.html`。
6. dashboard 和 report 中只展示摘要，详细逐项核验以 `2025计划与实际录取人数审计.html` 为准。
7. 后续等 2026 计划公布后，要把每个候选项放入同一张测算表：2025计划、2025实录、2025最低分位次、2026计划、计划增减、专业组代码、学费校区、招生章程风险。

阶段性结论：计划人数一般比实录人数更容易获取。实录人数不是所有学校都会按“陕西 + 物理类 + 普通批 + 专业”公开，因此要按“已取得/待核/不可公开获取”三档管理，不能因为暂时没有实录人数就把候选项删除，但小计划专业的风险要上调。
